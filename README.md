<h1 align="center">Qr Maker</h1>
<p align="center">Convert a text/link into a Qr code and save it.</p>

# Features
 - Graphic display.
 - Simple to use.

<br>

# How to use
 1. Install external modules.
 2. Launch [qr_maker.py](qr_maker.py).
 3. Enjoy !
